Virtual Keyboard – release notes

Congratulations! Your Virtual Keyboard extension has been successfully upgraded to 2.0 version and is up to date now :)
Restart your Opera browser to get it working, please.

Thank you for installing and using this extension. Please, read this short introduction to know all about the changes made from previous version.

I would like to invite you all to brand new support forums opened recently. You can post there your own ideas, ask me a question and report bugs (or specific websites where extension is not working for you). In group's blog I will be posting latest news about development process, announce new releases and updates so it is very good idea to subscribe to blog's feed.

Important!

  • This extension now works only with Opera 11.11 stable. Older versions are NOT supported anylonger. Upgrade your Opera if you need this extension.
  • This extension may, but might NOT work with new testing builds (including Opera Next) published in Desktop Team's blog.
  • This extension works ONLY with webpages' forms. It WON'T work with address bar, search box, etc. It is impossible to implement such feature right now.

What has been changed in this version?

  • 9 new keyboard layouts: Arabic (101), Arabic (102), Arabic (102) AZERTY, French, Italian, Norwegian, Portuguese, Spanish, Turkish Q. More to come!
  • Fixed Enter bug in ISO keyboard layouts.
  • Fixes for several websites reported by users (thank you!).
  • New feature: dedicated drag handler for those of you who type fast and find keyboard sticky ;)
  • Removed "Space" string from Space bar.

What still need to be polished?

  • Support for more keyboard layouts available (priority: high).
  • Problems specific to particular websites (priority: medium).
  • Translations for preferences page (priority: medium; need your help - I'll let you know soon).
  • New feature: optional numpad (priority: medium).
  • New feature: keyboard zooming (priority: medium).
  • Keyboard does not work in old–style framesets (priority: none - Opera bug).
