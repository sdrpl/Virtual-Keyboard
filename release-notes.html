Virtual Keyboard – release notes

Congratulations! Your Virtual Keyboard extension has been successfully upgraded to 2.1 version and is up to date now :)
Restart your Opera browser to get it working, please.

Thank you for installing and using this extension. Please, read this short introduction to know all about the changes made from previous version.

I would like to invite you all to extension's support forums. You can post there your own ideas, ask me a question and report bugs (or specific websites where extension is not working for you). In group's blog I post latest news about development process, announce new releases and updates so it is very good idea to subscribe to blog's feed.

Important!

  • This extension now works only with Opera 12.02 stable. Older versions are NOT supported anylonger. Upgrade your Opera if you need this extension.
  • This extension may, but might NOT work with new testing builds (including Opera Next) published in Desktop Team's blog.
  • This extension works ONLY with webpages' forms. It WON'T work with address bar, search box, etc. It is impossible to implement such feature right now.

What has been changed in this version?

  • New keyboard layouts: Chinese Bopomofo IME, Chinese ChaJei IME, Japanese (Hiragana), Japanese (Katakana), Serbian (Cyrillic), Serbian (Latin), Slovenian, Swiss French, Swiss German, Uzbek Cyrillic, Vietnamese.
  • Greek translation (thanks to Άγγελος Δούνδης).
  • Ukrainian translation (thanks to Askold Uaixovich).
  • Fixed show on double click (thanks to Арсений Краснов).
  • Fixed layout selection dropdown bug.
  • Translation of toolbar button's tooltip.
  • CSS fix for Preferences/Release notes pages.

What still need to be polished?

  • Support for more keyboard layouts available (priority: high).
  • Problems specific to particular websites (priority: medium).
  • New feature: optional numpad (priority: medium).
  • New feature: keyboard zooming (priority: medium).
  • Translations for preferences page (priority: medium; need your help!).
  • Keyboard does not work in old–style framesets (priority: none - Opera bug).
